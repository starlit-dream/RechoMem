# Draft: RechoMem 实现规划

## Requirements (confirmed)

- 项目语言: Rust，异步运行时 Tokio
- L1 向量层: LanceDB 嵌入式（Rust 原生 crate，无需外部进程）
- L2 摘要层: SQLite，通过 sqlx 异步访问
- L3 原始层: 纯 JSONL，追加写入，offset 指针随机读取
- LLM 调用: OpenAI 兼容 HTTP API，通过 async-openai crate
- MCP 传输: stdio（兼容 Claude Desktop、Cursor 等本地客户端）

## Technical Decisions

- 向量嵌入: 通过 async-openai embeddings endpoint 生成，模型 text-embedding-3-small
- 动态语义分块: 异步后台任务，在对话间歇期运行，调用 LLM 识别语义块边界
- 混合检索: L1 向量相似度 + L2 实体反向索引，时间衰减重排
- 错误处理: thiserror 定义 domain 错误类型，anyhow 用于顶层传播
- 日志: tracing + tracing-subscriber
- 测试策略: 集成测试为主（tests/ 目录），单元测试覆盖纯函数逻辑

## MCP Tools（三个对外接口）

- search_memory_index(query: str) → Vec<BlockSummary>
- read_context_summary(block_id: str) → Summary
- drill_down_raw_logs(block_id: str) → RawLogs
